COVID-19 Detection from Chest X-Ray Images Using CNN

Personal Project November 2024 - December 2024

Developed a CNN model to identify COVID-19 infections from chest X-ray images, achieving high accuracy across training, validation, and test datasets.

Key Highlights:

  • Built and trained a CNN using Keras with TensorFlow backend
  • Performed data preprocessing and augmentation using TensorFlow 2.0
  • Implemented sequential model architecture with convolutional, pooling, and dense layers
  • Analyzed lung X-ray images from Kaggle dataset
  • Achieved 94.83% accuracy on test set

Results:

  • Training Set: 98.41% accuracy, 0.0490 loss
  • Validation Set: 97.51% accuracy, 0.0759 loss
  • Test Set: 94.83% accuracy, 0.1141 loss
